## Create Server

Add a new server to your Dokploy infrastructure.

### POST `/server.create`

<ParamField body="name" type="string" required>
  Name of the server
</ParamField>

<ParamField body="ipAddress" type="string" required>
  Server IP address
</ParamField>

<ParamField body="port" type="number">
  SSH port (default: 22)
</ParamField>

<ParamField body="username" type="string" required>
  SSH username for server access
</ParamField>

<ParamField body="sshKeyId" type="string">
  ID of the SSH key to use for authentication
</ParamField>

<ParamField body="serverType" type="string">
  Type of server: `deploy` or `build`
</ParamField>

### Authentication

Requires user authentication.

### Example Request

```bash
curl -X POST "https://your-dokploy-instance.com/api/server.create" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "name": "Production Server",
    "ipAddress": "192.168.1.100",
    "port": 22,
    "username": "deploy",
    "sshKeyId": "key_123",
    "serverType": "deploy"
  }'
```

### Example Response

```json
{
  "serverId": "server_abc123",
  "name": "Production Server",
  "ipAddress": "192.168.1.100",
  "port": 22,
  "serverStatus": "inactive",
  "serverType": "deploy",
  "createdAt": "2024-03-15T10:00:00.000Z"
}
```

## Setup Server Monitoring

### POST `/server.setupMonitoring`

Configure monitoring for a remote server.

<ParamField body="serverId" type="string" required>
  The unique identifier of the server
</ParamField>

<ParamField body="metricsConfig" type="object" required>
  Monitoring configuration (same structure as admin.setupMonitoring)
</ParamField>

### Example Request

```bash
curl -X POST "https://your-dokploy-instance.com/api/server.setupMonitoring" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "serverId": "server_abc123",
    "metricsConfig": {
      "server": {
        "refreshRate": 5,
        "port": 9090,
        "token": "secure-token",
        "urlCallback": "https://metrics.example.com/callback",
        "retentionDays": 30,
        "cronJob": "0 */5 * * * *",
        "thresholds": {
          "cpu": 80,
          "memory": 85
        }
      },
      "containers": {
        "refreshRate": 10,
        "services": {
          "include": [],
          "exclude": []
        }
      }
    }
  }'
```

## Get Server Metrics
