## What is Dokploy?

Dokploy is a free, self-hostable Platform as a Service (PaaS) that simplifies the deployment and management of applications and databases. Deploy any type of application and manage your infrastructure with a modern, intuitive interface.

<Accordion title="Self-hosted Alternative to Vercel/Netlify">

## Getting Started

<Steps>
  <Step title="Install Dokploy">
    Run the installation script on your VPS to set up Dokploy in minutes.
    
    ```bash
    curl -sSL https://dokploy.com/install.sh | sh
    ```
  </Step>
  
  <Step title="Create Your First Project">
    Log in to the dashboard and create a project to organize your applications and databases.
  </Step>
  
  <Step title="Deploy an Application">
    Connect your Git repository or Docker registry and deploy your first application with a few clicks.
  </Step>
</Steps>
